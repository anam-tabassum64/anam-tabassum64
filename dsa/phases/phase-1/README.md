# Phase 1 Index - Arrays

This folder organizes Phase 1 into a maintainable interview-prep layout.

## Phase goal

Build strong array fundamentals for coding interviews using C++17.

Phase 1 starts with direct traversal and gradually moves into the patterns that appear most often in product-based company interviews:

- Hashing
- Two pointers
- Prefix and suffix computation
- Sliding window
- Greedy array scans
- Sorting-based array reasoning
- Binary search on answer
