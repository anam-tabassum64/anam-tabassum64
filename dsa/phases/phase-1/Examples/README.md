# Phase 1 - Examples

## Example inputs to practice

- `[1, 2, 3, 1]` for Contains Duplicate
- `[1, 2, 3, 4]` for Product of Array Except Self
- `[0, 1, 0, 2, 1, 0, 1, 3, 2, 1, 2, 1]` for Trapping Rain Water

## Extra dry-run cases

### Contains Duplicate

- `[1, 2, 3, 4]` should return `false`
- `[0, 0]` should return `true`
- `[-1, -2, -1]` should return `true`

### Product of Array Except Self

- `[1, 2, 3, 4]` should return `[24, 12, 8, 6]`
- `[0, 1, 2, 3]` should return `[6, 0, 0, 0]`
- `[0, 0, 2, 3]` should return `[0, 0, 0, 0]`

### Trapping Rain Water

- `[0, 1, 0, 2]` should return `1`
- `[4, 2, 0, 3, 2, 5]` should return `9`
- `[1, 2, 3, 4]` should return `0`

## How to use examples

1. Write the brute force solution
2. Walk through the example manually
3. Optimize the solution
4. Compare the dry run with your code
