# Phase 6 - Stack

## Scope

Stack problems are often about matching, reversal, and maintaining monotonic structure.

### Topics

- Stack implementation
- Balanced parentheses
- Infix, postfix, prefix
- Next greater element
- Next smaller element
- Stock span
- Largest rectangle
- Min stack
- Monotonic stack

## Practice direction

- Easy: balanced parentheses, stack simulation
- Medium: next greater element, stock span
- Hard: largest rectangle, monotonic stack problems
