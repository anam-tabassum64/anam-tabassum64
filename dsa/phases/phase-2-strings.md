# Phase 2 - Strings

## Scope

Strings are arrays of characters with extra care around frequency, order, and matching.

### Topics

- Character arrays
- Frequency counting
- Palindrome
- Reverse words
- Anagrams
- Longest substring
- Pattern matching
- KMP
- Rabin-Karp
- Z algorithm
- Rolling hash
- Trie basics

### Patterns

- Sliding window
- Two pointers
- Hashing
- Trie

## Study notes

- Start with brute force string scanning
- Use hashing for counting and lookup
- Use sliding window for substring problems
- Use trie when many prefix queries appear

## Practice direction

- Easy: palindrome check, reverse string, frequency count
- Medium: anagram grouping, longest substring without repeat, reverse words
- Hard: KMP, Z algorithm, rolling hash
