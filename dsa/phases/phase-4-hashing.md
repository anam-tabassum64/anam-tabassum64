# Phase 4 - Hashing

## Scope

Hashing helps with fast lookups, frequency counting, and prefix-based subarray problems.

### Topics

- `unordered_map`
- `map`
- `unordered_set`
- `set`
- Frequency counting
- Pair sum
- Longest consecutive sequence
- Subarray sum equals K
- Prefix XOR

## Key ideas

- Use hash maps when lookup speed matters
- Use ordered maps or sets when sorted order matters
- Prefix sums often pair well with hashing

## Practice direction

- Easy: frequency count, pair sum
- Medium: longest consecutive sequence, subarray sum K
- Hard: prefix XOR variants, hash-based optimization problems
