# Phase 16 - Advanced Data Structures

## Scope

This phase introduces advanced tools used for range queries, indexing, and performance-heavy problems.

### Topics

- Disjoint set union
- Segment tree
- Lazy propagation
- Fenwick tree
- Sparse table
- Binary lifting
- Euler tour
- Heavy-light decomposition
- Ordered set
- Treap

## Practice direction

- Easy: DSU basics, prefix queries
- Medium: Fenwick tree, sparse table, binary lifting
- Hard: segment tree with lazy propagation, heavy-light decomposition, treap-based problems
