## Example starting point

### Array Traversal

- Visit each element once from left to right or right to left
- Use a single loop
- Time complexity is `O(n)`
- Space complexity is `O(1)`

## Practice direction

- Easy problems: print, sum, min, max, reverse, count
- Medium problems: sorted check, prefix sum, rotate, move zeroes
- Hard problems: maximum subarray, trapping rain water, product except self
