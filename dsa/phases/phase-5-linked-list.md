# Phase 5 - Linked List

## Scope

Linked lists teach pointer manipulation and structural thinking.

### Topics

- Singly linked list
- Doubly linked list
- Circular linked list
- Reverse list
- Middle node
- Detect cycle
- Remove cycle
- Merge lists
- Sort list
- Intersection
- Clone linked list
- LRU cache

### Patterns

- Fast and slow pointer
- Dummy node
- Reverse pointer

## Practice direction

- Easy: traversal, insertion, deletion
- Medium: reverse, middle node, cycle detection
- Hard: merge sort on list, intersection, LRU cache
