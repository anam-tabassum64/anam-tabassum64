# Phase 3 - Examples

## Example inputs to practice

- `[1, 3, 5, 7, 9]` for Binary Search
- `[4, 5, 6, 7, 0, 1, 2]` for Search in Rotated Sorted Array
- `[3, 6, 7, 11]` with `h = 8` for Koko Eating Bananas

## How to use examples

1. Write the brute force version
2. Identify the monotonic property
3. Apply binary search
4. Verify with a dry run
