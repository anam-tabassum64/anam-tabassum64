# Phase 14 - Dynamic Programming

## Scope

DP solves overlapping subproblems by storing results and reusing them.

### Topics

- Memoization
- Tabulation
- Space optimization
- Fibonacci
- Climbing stairs
- House robber
- Knapsack
- Coin change
- LCS
- LIS
- Edit distance
- Matrix chain multiplication
- Partition DP
- Interval DP
- Digit DP
- Bitmask DP
- Tree DP
- DP on graphs

## Practice direction

- Easy: Fibonacci, climbing stairs
- Medium: house robber, knapsack, LIS
- Hard: LCS, edit distance, interval DP, digit DP, bitmask DP
